# 01. Fundamentos de Python

## 📖 Descripción

Esta sección cubre los conceptos fundamentales de Python, incluyendo la sintaxis básica, variables, tipos de datos y operadores. Es el punto de partida perfecto para comenzar tu viaje en Python.

## 🎯 Objetivos de aprendizaje

Al completar esta sección, serás capaz de:

- Entender la sintaxis básica de Python
- Trabajar con diferentes tipos de datos (números, strings, booleanos)
- Usar variables y constantes correctamente
- Aplicar operadores aritméticos, de comparación y lógicos
- Escribir y ejecutar programas básicos de Python

## 📚 Contenido

### 1. Introducción a Python
- ¿Qué es Python?
- Ventajas y aplicaciones
- Instalación y configuración

### 2. Sintaxis básica
- Comentarios
- Indentación
- Convenciones de nomenclatura (PEP 8)

### 3. Variables y tipos de datos
- Números (int, float, complex)
- Strings (cadenas de texto)
- Booleanos
- Conversión de tipos

### 4. Operadores
- Aritméticos (+, -, *, /, //, %, **)
- De comparación (==, !=, <, >, <=, >=)
- Lógicos (and, or, not)
- De asignación (=, +=, -=, etc.)

### 5. Entrada y salida básica
- Función `print()`
- Función `input()`
- Formateo de strings

## 💻 Ejemplos de código

```python
# Variables y tipos de datos
nombre = "Ana"
edad = 25
altura = 1.65
es_estudiante = True

print(f"Hola, soy {nombre}, tengo {edad} años y mido {altura}m")

# Operadores
a = 10
b = 3
print(f"Suma: {a + b}")
print(f"División entera: {a // b}")
print(f"Resto: {a % b}")

# Entrada de usuario
nombre_usuario = input("¿Cuál es tu nombre? ")
print(f"¡Hola, {nombre_usuario}!")
```

## 🏋️ Ejercicios prácticos
